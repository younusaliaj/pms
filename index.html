Project Description

The Process Management Simulation System is an Operating System based project. It demonstrates how an operating system creates, schedules, manages, blocks, resumes, and terminates processes. The project also shows Process Control Block, Ready Queue, CPU Scheduling, Gantt Chart, Waiting Time, and Turnaround Time.
